Apply to become an academy prototypes

This is where the Department for Education Service Delivery Division (SDD) Apply to become an academy (A2B) service prototypes live.

  • About this prototype

    This prototype demonstrates how we might allow separate schools to apply to join or form a single trust individually.

    You can:

    1. Create an application to join a trust
    2. Create a second application to join the same trust
    3. Create a single application for multiple schools to form a trust
    4. Add a school to a form a trust application
    5. Add more contributors to a school on an application to form a trust.

  • This prototype also simulates how we might allow the user to create a single application to join or form a trust, with each school providing their details.

    We simulate how an 'application owner' can:

    1. Create the application
    2. Add a school to the application
    3. Add a contributor from a school after it has been added to the application

Sprint 20

3 Jan - 16 Jan 2023

  • Complete application

    The user signs out and then returns to the application after the application's declaration has been completed by the chair.

  • Cancelled application

    The user signs out and then returns to the application after the application has been cancelled by another contributor.

  • Cancel the Application

    Users are able to cancel an application from the detail component at the bottom of the application overview.

  • Multiple applications

    Users can have more than one application at a time. They can also sign out should they need to create or continue an application from another DfE Sign In account.

  • Report a problem, get help, or leave feedback

    A form linked from the phase banner which allow users to get in contact if they experience problems, need help, or wish to leave feedback.
